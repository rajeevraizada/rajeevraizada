Math Crush!

How to play:


Levels range from addition up to calculus.

Soundtrack: "That's Mathematics", written, performed and
generously made public domain by Tom Lehrer.

Click here to start game



Short link to reach this game:
tinyurl.com/rajmathgame

Written by Raj Raizada: rajeev dot raizada at gmail dot com
Using libraries: p5.play, p5js, lodash, and numbers.js.
Source code available on Github, here.

Known bugs: touchscreen responses might not work on Firefox for Android, or on Chromebooks. On Chromebooks, use mouse trackpack instead of touchscreen.
Rarely: gameplay can reset to menu-screen for no apparent reason. Still working on tracking down that one!
