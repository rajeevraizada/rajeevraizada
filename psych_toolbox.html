Some tutorial Matlab Psychophysics Toolbox programs

The Matlab Psychophysics Toolbox, written by David Brainard and Denis Pelli, is a wonderful package for presenting stimuli. It allows Matlab programs to call low-level videocard and timing routines which provide extremely accurate control. It runs on Mac OS 9 and Windows, with Mac OS X and Linux on the way soon. I have used it for presenting visual, auditory and cross-modal stimuli, for psychophysical testing and for fMRI presentation.

Below are some programs that show how to present flickering log-polar radial checkerboards, rotating wedges and expanding rings using Colour Look-Up Tables (CLUTs). The nice thing about CLUTs is that they let you change what's on the screen without needing to do any time-consuming re-drawing of pixels. You can just change the colour of the pixels that are already there, in less than the blink of an eye. These programs also show how to present stimuli with frame-accurate timing. Please feel free to mail me with any questions or comments.
  • demo_clut_checkerboards.m - Run this program to see the flashing checkerboards in action. To run it, you also need the .mat file containing the matrix that defines the checkerboards, clut_check_pic.mat, and the program that makes the CLUTS: make_cluts.m

  • make_check_for_clut.m - This program shows how the matrix stored in clut_check_pic.mat was made. If you download that .mat file then you don't need to run this program. However, if you're curious to see how to make a log-polar radial checkboard, then take a look!
If you want to see more that the PsychToolbox can do, then a very good tutorial program is MinimalExample.m, written by Daniel Shima.
